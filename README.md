# PuzzleAdventure
Unity拼图冒险游戏

## 🎮 项目简介

这是一个Unity开发的拼图收集冒险游戏，玩家需要在地牢场景中探索并收集所有拼图来完成游戏。

## ✨ 已实现功能

### UI系统
- ✅ 游戏开始菜单界面
- ✅ 拼图收集进度UI显示
- ✅ 交互提示系统（"按E拾取"）
- ✅ 暂停菜单
- ✅ 胜利界面

### 场景系统
- ✅ 地牢场景基础布局
- ✅ 灯光系统设置
- ✅ 场景碰撞体设置
- ✅ 场景美化优化

### 视觉效果
- ✅ 火炬火焰粒子效果
- ✅ 灯光闪烁效果
- ✅ UI动画效果

### 脚本说明

所有脚本位于 `Assets/Scripts/` 目录：

- **UIManager.cs** - UI管理器
- **InteractionPrompt.cs** - 交互提示系统
- **PuzzleProgressUI.cs** - 拼图进度UI
- **PauseMenuController.cs** - 暂停菜单
- **VictoryScreenController.cs** - 胜利界面
- **SceneSetup.cs** - 场景设置
- **TorchFireEffect.cs** - 火炬火焰效果
- **GameManager.cs** - 游戏管理器（已更新）
